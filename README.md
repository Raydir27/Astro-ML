# Astro-ML

This repository contains code, models, and data for the Astro-ML project.

## Structure
- `src/` - Source code and notebooks
- `Data/` - Data files (large files tracked with Git LFS)
- `Models/` - Model weights and encoders

## Notes
- Large files (e.g., .fits, .csv, .parquet) are tracked with Git LFS.
- Some data files are excluded from version control for size reasons.

## Getting Started
1. Clone the repository:
   ```sh
   git clone https://github.com/Raydir27/Astro-ML.git
   cd Astro-ML
   git lfs pull
   ```
2. Install dependencies:
   ```sh
   pip install -r requirements_working.txt
   ```
3. Explore the notebooks in `src/` for examples and experiments.
